# README.md

# Audio material for *Understanding Linux: The Kernel Perspective*

This repository contains the encrypted audio files accompanying the book  
*Understanding Linux: The Kernel Perspective*.

Readers who purchased the book can access the audio material using the 
password provided in **Appendix 5**.


## Purchase Links

- **Leanpub:**  
  https://leanpub.com/linuxkernel

- **Amazon Kindle:**  
  https://www.amazon.com/dp/B0FBZQJ718

- **Paperback:**  
  https://www.amazon.com/dp/B0FCBKCGGC


## How to extract the audio material

1. Download or clone this repo.
2. Run the following command:

```bash
$ ./extract_audio.sh
```

3. When prompted, enter the password from **Appendix 5** of the book.
